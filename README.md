# 販売アシスト1号 コア版

請求書・領収証発行、顧客・商品マスター管理を行うAndroidアプリケーション

## 概要

**h-1-core**は、小規模事業者向けの販売管理アプリケーションです。
請求書や領収証の発行、顧客・商品マスターの管理など、基本的な販売業務をサポートします。

## 主な機能

- ✅ **請求書発行**: 請求書の作成・編集・PDF出力
- ✅ **領収証発行**: 領収証の発行・印刷
- ✅ **顧客管理**: 顧客情報の登録・編集・削除
- ✅ **商品管理**: 商品マスターの管理
- ✅ **伝票履歴**: 過去の請求書・領収証の検索・閲覧
- ✅ **PDF生成**: 請求書・領収証のPDF出力

## ダウンロード

最新版のAPKファイルは[Releases](https://github.com/krasherjoe/h1-core/releases)からダウンロードできます。

### インストール方法

1. [Releases](https://github.com/krasherjoe/h1-core/releases)から最新のAPKをダウンロード
2. Androidデバイスで「提供元不明のアプリ」のインストールを許可
3. ダウンロードしたAPKファイルをタップしてインストール

## 動作環境

- **OS**: Android 5.0 (API Level 21) 以上
- **ストレージ**: 100MB以上の空き容量

## 技術情報

- **フレームワーク**: Flutter 3.44.0
- **言語**: Dart 3.12.0+
- **データベース**: SQLite (sqflite)
- **PDF生成**: pdf, printing パッケージ
